# Instalasi

**Setup biasa Untuk jalankan bot**

```
$ cd ./your-path
$ npm install
$ npm start
```

**Setup rekomendasi ( Jika error )**

```
$ cd ./your-path
$ yarn --no-bin-links
$ npm start
```

**Setup dengan git**

```
$ git clone https://github.com/VendettaLovee/OblivionX-Wa
$ cd ./OblivionX-Wa
$ npm install
$ npm start
```
